## 🌟 Overview

Amdox Technology is a high-fidelity, highly interactive frontend enterprise platform designed to showcase modern, premium web development. It features a complete corporate ecosystem including secure authentication, synchronized profile management, interactive Kanban task boards, contactless QR attendance, and an integrated settings environment.

Built entirely with **Vanilla JavaScript, HTML5, and CSS3**—this project requires zero build steps, bundlers, or frameworks. It runs flawlessly on any local machine, right out of the box.

## ✨ Key Technical Features

- **Global Diagnostic Engine v3.2**: A robust custom JavaScript engine ensuring 100% interactive reliability, state persistence, and OS-agnostic navigation.
- **Glassmorphism UI System**: A breathtaking dark-mode interface featuring dynamic layered depth, backdrop-blurs, and vibrant accent gradients (Magenta & Orange).
- **Persistent Local State**: State management across multiple HTML files using `localStorage` (with graceful fallback session handling).
- **Global Profile Synchronization**: "Amdox Identity Engine" dynamically updates user avatars, initials, and names across all dashboards and modal windows simultaneously.
- **Interactive Feedback Loop**: Comprehensive "Toast" notification system and physical button-press scaling for premium UX.

## 🛠️ Module Breakdown

1. **🔒 Enterprise Authentication (`login.html` & `register.html`)**
   - Premium login and registration flow with password visibility toggles.
   - Intelligent profile capture.

2. **🏠 Executive Dashboard (`index.html`)**
   - Corporate identity showcase, global presence statistics, and hero entry points.

3. **📋 Dynamic Kanban Board (`kanban.html`)**
   - Drag-and-drop task management interface.
   - Interactive task creation modal with priority indexing.

4. **📅 Contactless Attendance (`attendance.html`)**
   - Real-time digital clock and active session tracking.
   - Dynamic QR Code generation (Powered by QRCode.js) for mobile check-ins.

5. **🔍 Asset Scanner (`scanner.html`)**
   - High-fidelity simulated camera scanner with flash-feedback.
   - Project/Asset resolution routing.

6. **⚙️ Profile Configurations (`settings.html`)**
   - Instant-switching tabbed interface.
   - Profile updating, simulated 2FA toggles, and notification preferences.

## 🚀 How to Run Locally

You do not need Node.js, npm, or any server configurations to run this platform.

1. **Download/Clone the repository**:
   ```bash
   git clone https://github.com/yourusername/amdox-technology.git
   ```
2. **Navigate** to the project folder.
3. **Double-click** `login.html` to open it in your default browser (Chrome recommended).
4. Enter any Name/Email to instantly access the platform!
